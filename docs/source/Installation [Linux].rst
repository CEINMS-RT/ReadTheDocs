Install script
++++++++++++++

The script that automates the process of setting up CEINMS-RT on a Linux system resides in the core repository. 
From the core repo's `homepage <https://github.com/CEINMS-RT/ceinmsrt-core-cpp>`_ on GitHub, 
click on the :guilabel:`Code` button, and then copy the clone URL to the clipboard.
Clone the repository to a location of your choice.
After cloning you execute the following commands:

.. code-block:: shell-session

   chmod +x script/install-linux.sh
   ./script/install-linux.sh
